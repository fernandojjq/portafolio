Aplicación móvil de banca en línea desarrollada para la Universidad Tecnológica de Panamá.

## 🎓 Conclusión del Curso

Este curso ha sido fundamental para adquirir los conocimientos técnicos actuales de la industria móvil. El aprendizaje clave se divide en tres pilares principales:

1. **Desarrollo con Kotlin:**
   * **Seguridad y Sintaxis:** Dominio de características como *Null Safety* para erradicar errores de tipo nulo en tiempo de ejecución.
   * **Eficiencia:** Creación de funciones de extensión y manejo de tareas en segundo plano a través de *Corrutinas* para evitar bloqueos en el hilo principal de la interfaz de usuario.

2. **Interfaces de Usuario con Jetpack Compose y Material:**
   * **Paradigma Declarativo:** Construcción ágil de interfaces que responden automáticamente a los cambios de estado en tiempo real.
   * **Diseño Estandarizado:** Adopción de guías y componentes visuales de **Material Design 3** para asegurar consistencia estética y de usabilidad.

3. **Automatización y Compilación con Gradle y AGP:**
   * **Gestión de Proyectos:** Administración eficiente de dependencias de terceros y plugins en el ecosistema Android Studio.
   * **Configuración del Build:** Optimización de tiempos de compilación utilizando sistemas de caché y la configuración de variantes (*build types* y *flavors*) para diferentes entornos.
